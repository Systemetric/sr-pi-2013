devices Package¶

devices Package¶

Devices that form part of a SR-less robot

These variants of the devices are used when we strip out the SR robot of any SR specific parts and use our own bits and pieces instead. Use these in the Raspberry Pi robots.

sendCommand(command, timeout=5.0)[source]¶

Sends command to mbed.

Sends a single command to the mbed, waits for the response and returns it. Command cannot contain linebreaks.

Parameters:
  • command – The single line command to be used. No linebreak necessary.
  • timeout (float or None) – The amount of time to wait for a response before giving up
Returns:

str – the mbed response
Raises :

ValueError – when command contains a linebreak
Raises :

RuntimeError – when mbed still has unread data in its buffer
